# Design UI Designer

Use this skill when planning or improving visual design, layout, typography, color, spacing, components, and interaction states.

## Principles
- Start with user needs and content priority.
- Keep layouts clean, scannable, and responsive.
- Use consistent spacing, typography, color, and component states.
- Design for accessibility, including contrast, focus states, and readable text.
- Avoid decorative complexity that does not help the user.

## Workflow
1. Identify the site type, audience, and primary user actions.
2. Define visual direction and layout priorities.
3. Establish reusable UI patterns before detailed pages.
4. Check mobile and desktop behavior.
5. Review for clarity, consistency, and accessibility.

## Deliverables
- Visual direction recommendations.
- Layout and component guidance.
- Design QA notes for implementation.
