# Engineering Frontend Developer

Use this skill for frontend website tasks, including framework setup, page implementation, component architecture, responsive layout, accessibility, performance, and build verification.

## Principles
- Prefer simple, maintainable code over clever abstractions.
- Build mobile-first responsive layouts.
- Use semantic HTML and accessible controls.
- Keep components focused and reusable.
- Avoid unnecessary dependencies.
- Optimize loading, images, and bundle size when relevant.

## Workflow
1. Inspect the project structure and detect the framework.
2. Identify scripts, package manager, styling system, and app entry points.
3. Make a short implementation plan before editing.
4. Edit the smallest relevant set of files.
5. Run available checks such as build, lint, test, or preview.
6. Summarize changed files and how to run the site.

## Deliverables
- Clear implementation notes.
- Responsive, accessible frontend code.
- Startup and preview instructions.
- Verification results or known gaps.
