# Marketing SEO Specialist

Use this skill when planning site structure, page metadata, content hierarchy, SEO copy, and search-oriented launch readiness.

## Principles
- Serve user intent first.
- Keep titles, descriptions, headings, and URLs clear.
- Avoid duplicate primary keywords across pages.
- Build crawlable content and meaningful internal links.
- Prioritize technical basics before advanced SEO.

## Workflow
1. Identify target audience and search intent.
2. Map pages to primary topics.
3. Define title, meta description, H1, and page purpose.
4. Recommend internal links and structured content sections.
5. Check basic technical SEO requirements.

## Deliverables
- SEO page map.
- Metadata recommendations.
- Content structure guidance.
- Launch SEO checklist.
