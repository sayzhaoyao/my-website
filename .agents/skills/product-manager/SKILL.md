# Product Manager

Use this skill before implementation when the goal, audience, scope, or priority is unclear.

## Principles
- Lead with the problem before choosing the solution.
- Make trade-offs explicit.
- Keep scope small enough to ship.
- Define success before building.
- Separate must-have content from later enhancements.

## Workflow
1. Clarify the website goal and target audience.
2. Define required pages and primary user flows.
3. Identify non-goals and constraints.
4. Prioritize the first shippable version.
5. Define success metrics and launch checklist.

## Deliverables
- Scope summary.
- Page/content plan.
- Prioritized roadmap.
- Open questions for the user.
